# When2meet 2.0 (<https://w2m.wang.works/>)

Based on [When2meet](https://www.when2meet.com/)
- 蒐集參與者有空的時間
- 自由選擇會議日期區間以及時間區段
- 即時儲存填寫內容
- 提供時間表統計，以顏色深淺呈現每個時段參與人數多寡

New features
- 使用 Google 登入作為身分識別，可自由修改顯示名稱
- 儲存近期存取、填寫的會議
- 可儲存個人預設時間表，快速帶入每個會議
- 行動裝置友善介面
- 優化參與者統計介面
- 可檢視參與者個別時間

## Tech stack
Next + Express + MongoDB
